Ultima actualizacion: junio de 2026

2. Alcance

Esta politica cubre la app Android, la app Windows y el sitio web de LocalClip.

3. Informacion tratada

LocalClip puede procesar el contenido que eliges sincronizar, como texto del portapapeles, capturas de pantalla y archivos, ademas de datos basicos necesarios para vincular tus dispositivos en la misma red local.

4. Finalidad

La informacion se utiliza para permitir la sincronizacion entre tus dispositivos, mostrar historial local, administrar archivos recibidos, mantener preferencias y operar funciones principales de la app.

5. Almacenamiento local

El historial, los ajustes y archivos recibidos se guardan localmente en tus dispositivos. LocalClip no requiere una cuenta en la nube para funcionar.

6. Servicios de terceros

La app puede usar servicios de terceros para anuncios, compras o distribucion mediante tiendas. Esos servicios tienen sus propias politicas.

7. Seguridad y eliminacion

Aplicamos medidas razonables para operar en red local. Puedes eliminar historial, archivos y configuraciones desde la app o el sistema operativo.
