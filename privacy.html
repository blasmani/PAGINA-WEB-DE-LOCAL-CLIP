Que hace LocalClip

LocalClip permite sincronizar portapapeles, capturas de pantalla y archivos entre un dispositivo Android y una PC Windows vinculados por el usuario. La app funciona principalmente en la red local y usa permisos del sistema solo para las funciones que el usuario activa o utiliza.

Datos que puede procesar

La aplicacion puede procesar texto del portapapeles, imagenes, capturas de pantalla, archivos seleccionados por el usuario, nombres de archivos, tamanos, hashes tecnicos de verificacion, informacion de vinculacion local, configuraciones, idioma, historial local y estado de conexion.

Estos datos se usan para sincronizar contenido, verificar transferencias, evitar duplicados, mostrar historial y mantener la conexion entre dispositivos autorizados.

Almacenamiento

LocalClip puede guardar informacion localmente en Android y Windows, incluyendo configuraciones, historial, archivos recibidos, capturas recibidas y datos de vinculacion. El usuario puede eliminar datos desde la app o desde las herramientas del sistema operativo cuando corresponda.

Tecnologias y servicios

La app usa tecnologias estandar de Android, Windows, red local, codigos QR, WebSocket/TCP local, almacenamiento local y verificacion criptografica de integridad. Segun la version y disponibilidad, puede usar servicios de Google Play, Google Play Billing y Google Mobile Ads para compras, anuncios o funciones relacionadas.

La descripcion se mantiene general por seguridad y para proteger detalles tecnicos internos de implementacion.

Responsabilidad del usuario

El usuario es responsable del contenido que copia, envia, recibe, guarda o comparte con LocalClip. No debe usar la app para transferir contenido ilegal, no autorizado o que vulnere derechos de terceros.

Cambios

Estas politicas pueden actualizarse para reflejar cambios legales, tecnicos o funcionales. La version publicada en este sitio sera la referencia vigente.
