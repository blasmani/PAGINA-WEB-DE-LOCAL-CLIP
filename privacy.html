Politica de privacidad

Ultima actualizacion: 29 de junio de 2026

2. Alcance

Esta politica se aplica al uso de LocalClip en dispositivos Android y equipos Windows, asi como al sitio web oficial localclip-synchronization.com. LocalClip permite que el usuario vincule voluntariamente un dispositivo Android con una PC Windows para sincronizar texto del portapapeles, capturas de pantalla y archivos seleccionados por el usuario. Algunas funciones dependen de permisos del sistema operativo, de la red local, de la configuracion del dispositivo, de servicios de Google Play o de componentes propios del sistema Android o Windows.

Esta politica no cubre aplicaciones, servicios, sitios web o plataformas de terceros que el usuario pueda usar junto con LocalClip, como aplicaciones de mensajeria, navegadores, servicios de almacenamiento, tiendas de aplicaciones, servicios de anuncios, sistemas operativos, antivirus, firewalls o herramientas de administracion del dispositivo. Cada tercero puede aplicar sus propias politicas.

3. Informacion que LocalClip puede tratar

Segun las funciones que el usuario active o utilice, LocalClip puede tratar informacion del portapapeles, texto copiado, imagenes copiadas, capturas de pantalla, archivos seleccionados para enviar, archivos recibidos, nombres de archivos, tamanos de archivos, tipo aproximado de contenido, fechas de eventos, estado de transferencia, historial local, configuracion de idioma, preferencias de sincronizacion, estado del teclado LocalClip Board, informacion de vinculacion entre dispositivos, identificadores tecnicos locales, direccion de red local, estado de conexion, codigos QR de vinculacion, hashes o comprobaciones tecnicas de integridad, registros tecnicos basicos y datos necesarios para mostrar progreso, confirmar recepcion o evitar duplicados.

Cuando el usuario utiliza el teclado LocalClip Board, Android puede permitir que el teclado lea el texto que el usuario escribe o gestiona en el campo activo, conforme a los permisos y advertencias del propio sistema. LocalClip utiliza esa capacidad para prestar funciones de escritura, portapapeles y sincronizacion cuando el usuario lo activa. El usuario puede cambiar de teclado o desactivar LocalClip Board desde los ajustes del sistema Android.

Cuando el usuario activa funciones de capturas o portapapeles de imagenes, LocalClip puede detectar imagenes nuevas generadas o disponibles en ubicaciones permitidas por Android, procesarlas localmente, preparar una copia para sincronizacion y transferirla al dispositivo vinculado. El tratamiento se realiza para que la captura pueda estar disponible en el otro dispositivo, en el historial o en el portapapeles, segun corresponda.

4. Finalidades del tratamiento

La informacion se trata para permitir la vinculacion entre Android y Windows, sincronizar contenido autorizado por el usuario, transferir archivos, mostrar progreso, guardar archivos recibidos, mantener el historial local, mostrar miniaturas de capturas en Windows, colocar contenido en el portapapeles del dispositivo receptor, verificar que una transferencia se completo correctamente, evitar duplicados, restaurar estados de transferencia, mejorar estabilidad, diagnosticar errores, aplicar preferencias del usuario, gestionar idioma, mostrar avisos necesarios, permitir compartir el enlace de la aplicacion y cumplir obligaciones legales o de seguridad.

LocalClip no esta disenado para crear perfiles publicitarios basados en el contenido del portapapeles, capturas o archivos del usuario. El contenido que el usuario sincroniza pertenece al usuario o a quien corresponda legalmente; LocalClip lo procesa solo para prestar la funcion solicitada o activada por el usuario.

5. Base y consentimiento del usuario

El uso de LocalClip es voluntario. El usuario decide instalar la aplicacion, abrirla, conceder permisos, vincular dispositivos, activar o desactivar funciones, seleccionar archivos, usar el teclado LocalClip Board, activar sincronizacion de portapapeles o realizar transferencias. Cuando una funcion requiere permiso del sistema operativo, la aplicacion solicita o utiliza dicho permiso conforme a las reglas de Android o Windows. Si el usuario deniega un permiso, algunas funciones pueden no estar disponibles.

El usuario puede detener la sincronizacion, eliminar la vinculacion, cambiar permisos, desinstalar la aplicacion, borrar datos locales o dejar de usar LocalClip en cualquier momento desde la aplicacion o desde las herramientas del sistema operativo.

6. Almacenamiento local

LocalClip puede guardar informacion localmente en el dispositivo Android y en la PC Windows. Esto puede incluir configuraciones, historial, datos de vinculacion, archivos recibidos, capturas recibidas, miniaturas, estado de transferencia, registros tecnicos basicos y preferencias de usuario. La ubicacion exacta puede depender de la version del sistema operativo, permisos concedidos y configuracion de la aplicacion.

Los archivos recibidos se guardan en ubicaciones accesibles o administradas por la aplicacion, como carpetas de LocalClip en Android o Windows. El usuario es responsable de administrar, conservar, mover o eliminar los archivos recibidos. Si el usuario borra datos de la aplicacion, elimina carpetas, desinstala LocalClip o usa herramientas de limpieza, parte de esa informacion puede perderse.

7. Transferencia entre dispositivos

La sincronizacion de LocalClip esta orientada principalmente a la comunicacion entre dispositivos autorizados por el usuario dentro de la misma red local o entorno compatible. Para vincular los dispositivos se usa un mecanismo de autorizacion mediante QR u otro flujo equivalente. Una vez vinculados, Android y Windows pueden intercambiar mensajes tecnicos y contenido elegido o activado por el usuario para cumplir las funciones de portapapeles, capturas y archivos.

El usuario debe vincular solo dispositivos propios o dispositivos sobre los que tenga autorizacion. LocalClip no debe usarse para copiar, transferir, interceptar o conservar informacion de terceros sin permiso.

8. Servicios de terceros

Segun la version instalada, el pais, la disponibilidad y la configuracion del usuario, LocalClip puede integrar servicios de terceros como Google Play, Google Play Billing, Google Mobile Ads, Android System WebView, servicios del sistema operativo, herramientas de distribucion de Google Play o servicios equivalentes necesarios para anuncios, compras, actualizaciones, seguridad, medicion tecnica o funcionamiento basico. Dichos terceros pueden tratar informacion conforme a sus propias politicas.

LocalClip no controla las practicas de privacidad de Google, Microsoft, fabricantes de dispositivos, operadores de red, aplicaciones de destino donde el usuario pega contenido, ni otros servicios externos. El usuario debe revisar las politicas correspondientes si utiliza esos servicios.

9. Publicidad y compras

LocalClip puede mostrar anuncios o permitir opciones para quitar anuncios, de acuerdo con la version publicada y la configuracion del usuario. Los anuncios pueden ser servidos por proveedores externos y pueden implicar el tratamiento de identificadores publicitarios, informacion aproximada del dispositivo, idioma, pais, interacciones con anuncios u otros datos regulados por las politicas del proveedor y del sistema operativo.

Las compras, suscripciones o funciones premium, si existen, pueden gestionarse mediante Google Play Billing u otros sistemas autorizados. LocalClip no almacena directamente los datos completos de pago del usuario; dichos datos son procesados por la plataforma de pago correspondiente.

10. Seguridad

LocalClip aplica medidas razonables para reducir riesgos y verificar transferencias, como controles tecnicos de vinculacion, confirmaciones, comprobaciones de integridad, almacenamiento local y separacion de funciones. Sin embargo, ningun sistema es completamente seguro. La seguridad tambien depende del dispositivo del usuario, red local, contrasenas, permisos, sistema operativo, firewall, antivirus, apps instaladas y uso responsable.

El usuario debe mantener sus dispositivos actualizados, proteger el acceso fisico, evitar redes no confiables, no vincular dispositivos desconocidos y no transferir informacion sensible si no comprende los riesgos.

11. Conservacion y eliminacion

La informacion local puede conservarse mientras la aplicacion este instalada, mientras exista historial, mientras haya archivos recibidos, mientras se mantenga una vinculacion o mientras sea necesaria para prestar la funcion solicitada. El usuario puede borrar historial, eliminar archivos, desvincular dispositivos, revocar permisos, borrar datos de la app o desinstalar LocalClip. Al eliminar informacion local, algunas funciones pueden requerir nueva configuracion o nueva vinculacion.

12. Menores de edad

LocalClip no esta dirigido especificamente a menores de edad. Si un menor usa la aplicacion, debe hacerlo con autorizacion y supervision de su padre, madre, tutor o responsable legal, especialmente cuando transfiera archivos, textos, imagenes o informacion personal.

13. Derechos y solicitudes

Dependiendo de la legislacion aplicable, el usuario puede tener derechos de acceso, rectificacion, eliminacion, oposicion, limitacion o informacion sobre el tratamiento de sus datos. En la practica, gran parte de la informacion de LocalClip se conserva localmente en los propios dispositivos del usuario, por lo que el usuario puede gestionarla directamente desde la aplicacion, las carpetas de almacenamiento o los ajustes del sistema.

Para consultas adicionales, puede escribir a webbusinesstoolspro@gmail.com.

Esta Politica de privacidad puede actualizarse para reflejar cambios funcionales, tecnicos, legales, comerciales o de seguridad. La version publicada en este sitio sera la version vigente. El uso continuado de LocalClip despues de una actualizacion implica que el usuario conoce la version actual disponible.
