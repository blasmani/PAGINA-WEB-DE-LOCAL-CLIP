FAQ

Questions, answered.

Is LocalClip free to download?+

Yes. The Windows app is free to download and install from this page.

Do I need both apps?+

Yes — install the Windows app on your PC and use the LocalClip Android app to pair via QR.

Does it need an internet account?+

No. Transfers happen over your local network between your phone and PC.

Which Windows version is supported?+

The installer is a self-contained Windows x64 build and does not require installing .NET manually.
